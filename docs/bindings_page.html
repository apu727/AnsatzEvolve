C++ interface

For C++ usage it is best to use the class stateAnsatzManager directly. The C API is a wrapper around this class.

See also
main.cpp for example usage.

Python Bindings

These also wrap the class stateAnsatzManager. PyAnsatzEvolve.cpp contains the implementation of all functions exposed to python.

In order to compile you must specify: 

-DCOMPILE_PYTHON_BINDINGS=ON

when configuring. A detectable version of pybind11 is also needed. The output library is placed in the root of the repository if the install target is specified. 

cmake --build . --target install
